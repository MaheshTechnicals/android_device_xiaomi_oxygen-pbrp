# Device tree for Xiaomi Mi Max 2 (codenamed _"oxygen"_)

Kernel source: "https://github.com/Sanek-creed/kernel_pie.git"
==================================
## Device specifications

| Feature                 | Specification                     |
| :---------------------- | :-------------------------------- |
| CPU                     | Octa-core 2.0 GHz Cortex-A53      |
| Chipset                 | Qualcomm MSM8953 Snapdragon 625   |
| GPU                     | Adreno 506                        |
| Memory                  | 4 GB                              |
| Shipped Android Version | 7.1.1                             |
| Storage                 | 64/128 GB                         |
| MicroSD                 | Up to 256 GB                      |
| Battery                 | 5300 mAh (non-removable)          |
| Dimensions              | 88.7 x 174.1 x 7.6 mm             |
| Display                 | 1920x1080 pixels, 6.44 (~342 PPI) |
| Rear Camera             | 13 MP, Dual LED flash             |
| Front Camera            | 5 MP                              |
| Release Date            | May 2017                          |
